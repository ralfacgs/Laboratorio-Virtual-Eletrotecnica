Notas de Engenharia: Análise Vetorial CA

Este documento detalha a arquitetura matemática base implementada nos arquivos diagramas_fasoriais.html e soma_fasores.html.

1. Modelagem Vetorial e Conversão de Coordenadas

O simulador gera representações gráficas de grandezas senoidais no domínio dos fasores. O cálculo das componentes retangulares (Real e Imaginária) a partir da forma polar é feito através de trigonometria básica, permitindo a transição rigorosa em tempo real entre módulo/fase e o plano cartesiano.

// Conversão de Polar (Amplitude, θ) para Retangular (a + bj)

const rad = phase * Math.PI / 180;

const real = amplitude * Math.cos(rad);

const imag = amplitude * Math.sin(rad);

2. Rotação Temporal Sincronizada

Para conectar o conceito de ciclo trigonométrico com a evolução no tempo da senoide, o sistema utiliza a API nativa do navegador para incrementar o tempo t. O vetor (fasor) gira com velocidade angular ω, e sua ponta é perfeitamente alinhada com o eixo Y da senoide renderizada simultaneamente.

// Cálculo do ângulo instantâneo e seu valor na senoide

const currentAngle = phase0 + (freqAngular * t);

const valorInstantaneo = Vp * Math.sin(currentAngle);

3. Escalonamento Dinâmico Inteligente (Auto-Zoom)

Para evitar que os vetores ultrapassem os limites visíveis do Canvas — especialmente durante a soma vetorial, onde a Resultante pode ser substancialmente maior —, o algoritmo implementa um fator de escala dinâmico. O sistema detecta a maior grandeza presente na tela (V₁, V₂ ou VR) e adapta o "zoom" para que todos se encaixem perfeitamente.

// Fator de proporção para renderização vetorial

const maxVal = Math.max(a1, a2, aR);

const scale = (canvas.width / 2 - 60) / maxVal;

4. Regra do Paralelogramo e Interatividade

No módulo de Soma Vetorial, as projeções tracejadas são calculadas geometricamente espelhando as componentes de cada fasor. Além disso, a interatividade "Click & Drag" permite ao usuário arrastar a ponta do vetor. O código capta a coordenada do mouse no Canvas, inverte o eixo Y (que cresce para baixo em computação gráfica) e usa o Arco-Tangente bidimensional (Math.atan2) para recalcular a fase.

// Interação do Mouse para girar o vetor manualmente

let angle = Math.atan2(-y, x) * (180 / Math.PI);

document.getElementById('numPhase').value = Math.round(angle);

5. Ocultação Pedagógica (Cadeado do Professor)

Alinhado à filosofia didática do Lab AC Pro, o painel com o "Passo a Passo da Soma" (que realiza toda a mecânica matemática da transformação para retangular e soma algorítmica) é bloqueado por padrão. O sistema de autenticação, ancorado ao logins.js, exige as credenciais do professor para exibir a resolução em tela, incentivando que a turma resolva o problema analiticamente em caderno primeiro.
