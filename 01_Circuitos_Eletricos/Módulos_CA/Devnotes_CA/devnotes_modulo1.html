Notas de Engenharia: Sinais CA Básicos

Este documento detalha a arquitetura matemática base implementada no arquivo modulo1_sinais_ca_basico.html.

1. Modelagem da Onda Senoidal

O simulador gera uma representação vetorial em tempo real da grandeza elétrica. A equação fundamental que dita o comportamento do gráfico converte a frequência em Hz informada no painel para a velocidade angular da máquina geradora.

// Equação Base: v(t) = Vp * sin(2πft)

const omega = 2 * Math.PI * f;

const v_t = Vp * Math.sin(omega * tSec - phaseShift);

2. Base de Tempo Dinâmica (Zoom Time/Div)

Atuando como o seletor de "Time/Div" de um osciloscópio físico, o algoritmo permite ao usuário ajustar a escala do eixo X para exibir de 0.5 a 1.000 ciclos. A duração total da amostragem em tela é calculada em tempo real multiplicando-se a quantidade de ciclos exigida pelo Período elétrico.

const T = 1 / f; // Período de um ciclo

const totalDurationMs = cycles * T * 1000; // Janela total visível

3. Proteção contra Aliasing (Amostragem Inteligente)

Para garantir que a onda mantenha a curvatura perfeita independentemente do zoom aplicado, o sistema adapta a quantidade de pontos calculados (samples). O algoritmo aloca idealmente 50 pontos matemáticos por ciclo, mas implementa um teto máximo rígido de 5.000 pontos. Essa proteção impede que varreduras extensas (ex: 1.000 ciclos) travem a memória RAM do navegador e evita o fenômeno visual de subamostragem (Aliasing).

// Limita entre 150 (mínimo de fluidez) e 5000 (teto de hardware)

const samples = Math.min(5000, Math.max(150, Math.round(cycles * 50)));

4. Referência de Tensão Eficaz (True RMS)

Para consolidar o conceito de que a Corrente Alternada possui um valor constante equivalente em dissipação térmica (efeito Joule), o gráfico injeta uma reta de referência absoluta (Vrms). O valor RMS negativo é omitido propositalmente na versão atual do sistema, demonstrando pedagogicamente aos alunos que o cálculo eficaz (Root Mean Square - Raiz Quadrada da Média dos Quadrados) origina sempre uma grandeza física estritamente positiva.

5. Inspeção de Valores Instantâneos

Aproveitando os recursos interativos de *tooltips* com mode: 'dic_circuitos_eletricos', o eixo cruzado do mouse permite ao aluno inspecionar a coordenada exata em um dado milissegundo. Essa ferramenta comprova empiricamente que o valor instantâneo flutua freneticamente e cruza o zero, enquanto o valor eficaz (leitura do multímetro) se mantém como uma linha imperturbável sobre o tempo.

6. Motor de Animação e Deslocamento Temporal

Para o botão "Simular Fluxo de Tempo", a ferramenta utiliza a API requestAnimationFrame() nativa dos navegadores. Em vez de deslocar os vetores do gráfico destrutivamente, o algoritmo incrementa a constante de fase (phaseShift) da função seno. Isso cria uma ilusão óptica contínua e perfeita do tempo passando sem degradar os dados anteriores em memória.
