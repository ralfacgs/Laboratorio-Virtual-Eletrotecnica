Histórico de Atualizações

Notas de desenvolvimento e correções de engenharia da Calculadora Lab AC Pro.

v1.4.0 Junho de 2026

Histórico Interativo e Chave Bidirecional (P ⇄ R)

  • Painel de Histórico Lateral: Inclusão de um novo painel lateral para registrar os últimos 20 cálculos realizados, com interface rolável e botão de limpeza.
  • Reuso Inteligente de Dados: Botões interativos no histórico para copiar expressões originais ou resultados diretamente para o display. O sistema identifica expressões mistas e números complexos, adicionando parênteses protetores automaticamente para evitar falhas de sintaxe na próxima conta.
  • Alternância Polar/Retangular: Evolução do botão de conversão para atuar como uma chave bidirecional (P ⇄ R). Permite alternar qualquer resultado no display instantaneamente entre as notações trigonométricas e cartesianas.
  • Correção de Regex (Trigonometria): Solução do bug que causava a duplicação da variável deg (ex: cos(30 deg deg)) em operações mistas. A ordem de tradução na função preprocessMath() foi invertida para garantir a integridade matemática.
v1.3.0

Consolidação Mista e Otimização de Precisão

  • Suporte a Operações Mistas: Otimização do parser para permitir expressões que misturam fasores e formato retangular na mesma linha de cálculo, como (20∠30) + (20+5i).
  • Função cleanComplexRounding(): Implementação de algoritmo para eliminar resíduos binários de ponto flutuante gerados por conversões trigonométricas automáticas no JavaScript (ex: limpando lixos como 15.000000000001i para 15i).
  • Limpeza de Termos Nulos: Inclusão de filtro Regex pós-processamento para remover componentes complexos nulos redundantes, omitindo exibições como + 0i.
v1.2.0

Implementação de Entrada Fasorial Nativa

  • Entrada Direta (R∠θ): Adicionado o botão de ângulo (∠) no painel físico superior.
  • Motor Regex Adaptativo: Desenvolvimento de rotina dentro de preprocessMath() para converter dinamicamente strings do tipo fasor para coordenadas cartesianas puras: R · cos(θ) + j · R · sin(θ).
  • Sincronismo Angular: Amarração direta da conversão fasorial ao estado lógico do seletor DEG/RAD.
v1.1.0

Funções Inversas e Correção de UI

  • Trigonometria Inversa: Adicionados os escopos trigonométricos de arco (asin, acos, atan) vinculados ao cálculo de defasagem e fator de potência.
  • Módulo Absoluto (|x|): Inclusão da função de valor absoluto para extração rápida de magnitude de impedâncias.
  • Correção do Bug do Superscript: Substituição das entidades textuais corrompidas (como &supmin;¹) pela injeção da tag estrutural nativa <sup>-1</sup>.
v1.0.0

Lançamento Base (HP Prime Concept)

  • Layout de Categorias: Interface responsiva em blocos divididos inspirados no chassi da calculadora HP Prime.
  • Math.js Engine: Integração inicial com a biblioteca matemática para suporte algébrico nativo ao número imaginário i.
  • Conversor Polar (→ P∠): Rotina para extração e renderização visual instantânea de módulo e argumento de números complexos retangulares.
