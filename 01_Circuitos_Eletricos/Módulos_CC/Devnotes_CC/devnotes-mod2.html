Notas de Engenharia: Associação de Resistores

Este documento detalha a arquitetura matemática base implementada no arquivo modulo2_associacao_resistores.html.

1. Modelagem Matemática das Associações (Lei de Ohm)

O simulador computa dinamicamente a resistência equivalente ($R_{eq}$) para as duas topologias clássicas de circuitos baseados em corrente contínua. A partir desse valor e da tensão fornecida, determina-se o fluxo de cargas do circuito por meio da Primeira Lei de Ohm.

// Topologia 1: Circuito Série (Req = R1 + R2)

const reqSeries = r1 + r2;

const iSeries = v / reqSeries;

// Topologia 2: Circuito Paralelo (Req = (R1 * R2) / (R1 + r2))

const reqParallel = (r1 * r2) / (r1 + r2);

const i1 = v / r1;

const i2 = v / r2;

2. Divisor de Tensão Dinâmico (Malha Série)

No circuito série, a corrente é estritamente invariante ao longo de toda a malha, enquanto a diferença de potencial distribui-se de forma diretamente proporcional a cada componente ôhmico. O sistema calcula e plota essas parcelas em tempo real, validando pedagogicamente a Segunda Lei de Kirchhoff (Lei das Malhas).

// Quedas de Tensão parciais

const v1 = r1 * iSeries;

const v2 = r2 * iSeries;

// Validação: v1 + v2 deve convergir perfeitamente para v_total

3. Conservação de Carga e Divisor de Corrente (Nós Paralelos)

Ao ativar a malha paralela, o simulador demonstra fisicamente a Primeira Lei de Kirchhoff (Lei dos Nós). A corrente principal que sai do polo positivo da bateria subdivide-se inversamente à resistência de cada ramificação. Para traduzir isso de forma visual aos alunos, o motor gráfico renderiza fluxos de elétrons independentes com velocidades parametrizadas pelas intensidades locais de corrente ($I_1$ e $I_2$).

4. Dissipação de Potência por Efeito Joule (Gradiente Térmico)

Para consolidar o entendimento prático do efeito de aquecimento ôhmico, o sistema calcula continuamente a potência elétrica convertida em energia térmica ($P = I^2 \cdot R$). Com base nesses valores, os resistores sofrem uma alteração cromática dinâmica no Canvas, mudando de cor (cinza escuro → marrom → laranja → vermelho brilhante com efeito glow) para representar graficamente o estresse de potência e dissipação térmica do componente.

5. Arquitetura do Motor Elétron-Cinético (Offsets Independentes)

Diferente de simulações lineares convencionais, o script utiliza variáveis de deslocamento independentes (animMain, animB1, e animB2) incrementadas sob taxas proporcionais às correntes calculadas. A constante DOT_SPACING = 55 garante uma densidade enxuta, limpa e balanceada de partículas, permitindo que a divergência de velocidade e a divisão de cargas nos nós paralelos fiquem nitidamente visíveis em tela.
